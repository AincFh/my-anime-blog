# A.T. Field 前端视觉与交互设计升级计划

## 1. 核心设计哲学实现：双重世界

### 1.1 表世界（Public Interface）升级
- **背景系统优化**：
  - 添加黑色遮罩层，确保文字可读性
  - 实现视差滚动效果，背景滚动速度为前景50%
  - 优化背景图加载和切换动画

- **色彩体系重构**：
  - 主色：#ec4899（霓虹粉）
  - 辅色：#8b5cf6（赛博紫）
  - 文字色：#f8fafc（几乎纯白）+ #94a3b8（蓝灰）

- **毛玻璃效果升级**：
  - 优化玻璃材质：背景rgba(30, 41, 59, 0.6)，模糊16px
  - 添加悬浮效果：transform: translateY(-5px)，边框发光
  - 统一卡片样式和阴影效果

- **布局与视觉特效**：
  - 实现标题Glitch故障艺术效果
  - 采用非对称网格/瀑布流布局
  - 添加切角设计，类似EVA UI风格

### 1.2 里世界（Admin Interface）重构
- **视觉风格转换**：
  - 从NERV红色主题转换为Sakura-Soft樱花柔光风格
  - 背景：#fdf2f8（极浅粉）+ 漂浮彩色圆球
  - 色彩：主色#f472b6（樱花粉），文字色#374151（深灰）

- **组件设计升级**：
  - Soft UI风格：极大圆角（24px），弥散阴影
  - 左侧添加看板娘组件
  - 优化输入框样式：底部线条+focus粉色柔光

## 2. 动效与交互设计实现

### 2.1 页面转场效果
- 实现页面切换动画：当前页面下沉淡出，新页面上浮淡入
- 添加黑色切片过场效果，类似动漫转场

### 2.2 滚动反馈系统
- 元素进场动画：文章卡片按顺序依次飞入
- 顶部阅读进度条，随主题色变化

### 2.3 自定义光标与交互
- 实现自定义光标：默认空心圆环，hover时收缩为实心点
- 点击反馈：波纹扩散或粒子炸裂效果

### 2.4 音乐播放器组件
- 左下角旋转黑胶唱片图标
- 展开态：毛玻璃面板，显示歌词和频谱
- 路由切换时保持播放器不销毁

## 3. 响应式设计优化

### 3.1 移动端适配
- 取消视差滚动，背景图变暗20%
- 导航折叠为底部Dock栏
- 隐藏看板娘，节省空间

### 3.2 桌面端增强
- 启用所有特效和视差效果
- 优化大屏幕布局，增强沉浸感

## 4. 技术实现细节

### 4.1 CSS与样式优化
- 使用Tailwind CSS v4实现设计规范
- 优化CSS变量，支持主题切换
- 实现玻璃态和Soft UI的混合使用

### 4.2 动画与交互
- 使用Framer Motion实现所有动效
- 优化动画性能，避免过度渲染
- 实现流畅的状态过渡

### 4.3 组件架构
- 重构ThemeSwitcher，支持双重世界主题切换
- 实现可复用的玻璃态和Soft UI组件
- 优化组件通信和状态管理
